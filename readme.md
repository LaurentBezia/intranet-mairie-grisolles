# Intranet Mairie

Intranet interne pour une collectivité locale.

## Fonctionnalités
- Annuaire des agents
- Organigramme
- Communication interne
- Gestion des tâches
- Agenda partagé
- Connexion sécurisée

## Installation
1. Télécharger le projet
2. Ouvrir index.html
3. Ajouter vos graphismes dans /assets/images

## RGPD
- Accès par identifiant
- Données minimales
